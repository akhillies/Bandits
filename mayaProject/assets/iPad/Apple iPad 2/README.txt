2. HOW TO USE

We do our best to make all products simple to use. The name of the free file contains the product number and some description. For example mpm_F01_vray.MAX means the free product number 1 prepared for use in *.max scenes with vray rendering system. Some models are provided in different materials. All available variations are showed on previews. To get a model with another material do the following: select this model, choose the material you want from the material library or *.mat file and assign it. Note that materials are available only for 3dsmax scenes with scanline or vray rendering system.
